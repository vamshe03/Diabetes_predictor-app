# Diabetes_predictor-app


## 📌 Project Description
This project is a Machine Learning web application that predicts whether a person has diabetes or not based on medical input parameters.  
The model is built using **Logistic Regression** and deployed using **Streamlit**.

---

## 🚀 Features
- User-friendly web interface
- Real-time prediction
- Visualizations using Seaborn
- Trained ML model saved using joblib
- Easy deployment on Streamlit Cloud

---

## 🧠 Machine Learning Model
- Algorithm: Logistic Regression
- Dataset:  Diabetes Dataset
- Output: Binary classification (Diabetic / Non-Diabetic)

---

## 📊 Input Features
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Seaborn
- Matplotlib
- Joblib
